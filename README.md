## Rest Assured
Testing of Rest Services with Rest Assured <br/>
Test runs as own Application <br/>


### Technologies
Application: Spring, SpringBoot  <br />
Test: Rest-assured <br />



### Steps
##### Create jar, run jar
*gradle assemble*  <br />
*java -jar build/libs/jar-name.jar* <br />


##### Run with spring-boot-gradle-plugin
*gradle bootRun* <br />
